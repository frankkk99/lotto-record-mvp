# ระบบจดเลขเร็ว

เว็บแอพ MVP สำหรับบันทึกรายการเลขส่วนตัวแบบหน้างาน เหมาะกับผู้ใช้มือถืออายุประมาณ 30–50 ปี เน้นปุ่มใหญ่ จดเร็ว กันพลาด และสรุปยอดง่าย

> ระบบนี้เป็นเครื่องมือบันทึกข้อมูลส่วนตัวเท่านั้น ไม่มีระบบรับเงินออนไลน์ ไม่มีฝากถอน ไม่มีระบบจ่ายเงิน และไม่คำนวณอัตราจ่าย

## สิ่งที่ปรับปรุงล่าสุด

- เปลี่ยนแกนจาก **สลาก 6 หลัก** เป็น **จดเลข 1–3 หลักแบบหน้างาน**
- Flow ใหม่: `ชื่อลูกค้า → เลข → ประเภท → ยอด → บันทึก`
- เพิ่มประเภทหลัก:
  - 2 ตัวบน
  - 2 ตัวล่าง
  - 3 ตัวตรง
  - 3 ตัวโต๊ด
  - วิ่งบน
  - วิ่งล่าง
- เพิ่มแป้นตัวเลขใหญ่ กดง่ายบนมือถือ
- เพิ่มปุ่มยอดเร็ว `10 / 20 / 50 / 100 / 200 / 500`
- เพิ่มสถานะ `จ่ายแล้ว / ค้างจ่าย`
- เพิ่มปุ่ม `กลับเลข` สำหรับบันทึกรายการกลับอัตโนมัติ
- เพิ่ม Preview ก่อนบันทึก เพื่อลดการจดผิด
- ถ้าชื่อเดิม + เลขเดิม + ประเภทเดิม + สถานะเดิมซ้ำ ระบบจะรวมยอดให้เอง
- เพิ่มหน้ารายการ ค้นหาชื่อ/เลข/ประเภท เปลี่ยนสถานะ และลบแบบยืนยัน
- เพิ่มหน้าสรุปตามลูกค้า พร้อมยอดรวม จ่ายแล้ว ค้างจ่าย
- เพิ่มวางข้อความจาก LINE เช่น `พี่บอย 25 บน 50`
- ยังรองรับ Copy / Share / Export CSV

## จุดเด่นสำหรับผู้ใช้ 30–50 ปี

1. **เปิดมาแล้วจดได้ทันที** — ไม่ต้องเข้าเมนูซับซ้อน
2. **ปุ่มใหญ่** — เหมาะกับมือถือ ใช้งานตอนลูกค้ารอได้
3. **ตัวเลขใหญ่กลางจอ** — เห็นเลขชัดก่อนบันทึก
4. **ข้อมูลครบก่อนบันทึก** — ถ้าชื่อ เลข ประเภท หรือยอดไม่ครบ จะไม่ให้บันทึก
5. **รวมรายการซ้ำ** — ลดการมีรายการซ้ำหลายแถวโดยไม่ตั้งใจ
6. **สรุปตามลูกค้า** — เห็นยอดรวม ค้างจ่าย และจ่ายแล้วทันที

## รูปแบบข้อความ LINE ที่รองรับ

```txt
พี่บอย 25 บน 50
พี่บอย 25 ล่าง 50
แอน 123 โต๊ด 20
แม่แดง วิ่งบน 7 100
สมชาย 89 บน 100 ค้าง
```

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- localStorage สำหรับ MVP
- Vercel-ready deployment

## วิธีรันในเครื่อง

```bash

npm run dev
```

```bash
http://localhost:3000
```


## Roadmap รอบต่อไป

- เพิ่มปุ่มแก้ไขรายการ ไม่ใช่แค่ลบ/เปลี่ยนสถานะ
- เพิ่มหน้าแยกตามเลข เช่น เลข 25 มีใครบ้าง
- เพิ่มหน้าแยกตามประเภท เช่น 2 ตัวบน / 2 ตัวล่าง
- เพิ่มระบบล็อกงวด กันบันทึกผิดงวด
- เพิ่ม Supabase Auth และฐานข้อมูลจริง
- เพิ่ม Role: admin / staff / viewer
- เพิ่ม PDF export หรือใบสรุปลูกค้า
- เพิ่ม Audit log สำหรับการแก้ไข/ลบรายการ
- เพิ่ม Backup/restore รายงวด

## หมายเหตุด้านความปลอดภัย

- เวอร์ชันนี้เก็บข้อมูลใน browser localStorage เพื่อทดสอบ UX ก่อน
- ห้ามใช้เก็บข้อมูลจริงระยะยาวจนกว่าจะต่อฐานข้อมูลจริงและระบบล็อกอิน
- ไม่ควรเพิ่มระบบฝากถอน รับเงินออนไลน์ คำนวณอัตราจ่าย หรือจ่ายเงินในระบบ
